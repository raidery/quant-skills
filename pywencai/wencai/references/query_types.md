
## 估值筛选查询

### 低估值
```
市盈率小于20且市净率小于2的股票
市盈率小于30且ROE大于15%的股票
```

### 高成长
```
营收增长大于30%且净利润增长大于50%的股票
ROE大于20%且营收增长大于20%的股票
```

### 综合筛选
```
市盈率小于30且市净率小于3且ROE大于10%的股票
总市值大于100亿且ROE大于15%且市盈率小于25的股票
```

## 组合查询示例

### 价值投资筛选
```
市盈率小于20且ROE大于15%且营收增长大于10%的股票
```

### 成长股筛选
```
营收增长大于30%且净利润增长大于50%且ROE大于20%的股票
```

### 技术面+基本面
```
股价站上60日均线且ROE大于15%的股票
MACD金叉且市盈率小于30的股票
```

## 使用技巧

1. **精确表达** - 使用明确的数值和条件
2. **组合条件** - 用"且"连接多个条件
3. **时间范围** - 指定"今日"、"近5日"、"近一年"等
4. **排序优化** - 配合 sort_key 参数获得更好的结果
